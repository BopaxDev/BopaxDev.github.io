Data Cleaning and Analysis¶

Overview¶

Through this Jupyter Notebook, we are going to analyse the financial datasets prepared in Excel, with all the data imported from the Bloomberg Terminal.

The Excel Workbook is located in the data folder of the main project with the name: Bloomberg_Raw_Data.xlsx

Before having a look to the results coming from cleaning and analysis result, it is important to refer the scripts provided through the folder src of the main project, all the Classes and Methods defined there help the user to have a clear view of the results.

In the aforementioned scripts, also a Yahoo Finance Datasets cleaner is provided.

Due to the scant amount of the data, Bloomberg Terminal source is currently preferred.

Bloomberg Source¶

With the following code blocks, three different financial instruments will be presented. In particular:

  • SPX
  • NDX
  • MSCI US Equity

As could be seen, from the first block, this Jupyer Notebook is actually able to have a look to the entire Excel Workbook, providing to the user different choices and decide which financial asset to analyse.

In this specific scenario, we are going to analyse, for each of the financial instruments before mentioned: different statistical indicators and plots.

These would be needed to extract eventual and significant information for the Portfolio Optimization Model, which is the objective of the entire project.

import matplotlib.pyplot as plt

fig, axs = plt.subplots(nrows=2, ncols=3, figsize=(20, 15))
fig.suptitle(f"Plot Analysis - {asset} - From {start_date} to {end_date}", fontsize=32)

axs[0, 0].set_title("Close Prices Plot", fontsize=14)
axs[0, 0].plot(bloomberg_asset.dates(), bloomberg_asset.close_prices(), color="green")
axs[0, 0].tick_params(axis="x", labelrotation=45)
axs[0, 1].set_title("Returns Distribution", fontsize=14)
axs[0, 1].hist(bloomberg_asset.returns(), bins=50, color="red")
axs[0, 2].set_title("Volume Distribution", fontsize=14)
axs[0, 2].hist(bloomberg_asset.volumes(), bins=50, color="blue")

axs[1, 0].set_title("Aggregated Call And Put Options Volume Distribution", fontsize=14)
axs[1, 0].hist(bloomberg_asset.call_options_volume(), bins=50, color="green", alpha=0.4)
axs[1, 0].hist(bloomberg_asset.put_options_volume(), bins=50, color="red", alpha=0.4)
axs[1, 1].set_title("Call Options Volume Distribution", fontsize=14)
axs[1, 1].hist(bloomberg_asset.call_options_volume(), bins=50, color="green", alpha=0.5)
axs[1, 2].set_title("Put Options Volume Distribution", fontsize=14)

Conclusion¶

The deep reinforcement learning field applied to finance requires us to highlight the statistical and financial properties of a financial asset.

By understanding these properties, such as volatility and distributional structures, these type of algorithms can effectively navigate and, potentially exploit, complex market dynamics (This is what we hope).

Moreover, incorporating empirical stylized facts, such as fat tails in return distributions or volatility clustering, enhances the robustness and realism of these financial models.
